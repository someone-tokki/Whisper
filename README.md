# Whisper

Whisper 是一个为 iOS 设计的 ASMR 本地播放方案。它使用 SwiftUI 和 AVFoundation 构建，重点解决 ASMR 作品在 iPhone / iPad 上的本地整理、连续播放、字幕观看、后台播放和睡前收听体验。

项目当前仍处于早期版本，但已经可以作为一个可运行的原生 iOS 播放器使用。

## 项目定位

- 面向 ASMR、音声、助眠视频和带字幕内容的本地播放。
- 不依赖云端账号，不上传媒体文件，直接读取设备本地文件。
- 优先适配 iOS Files 工作流：把作品放进 `我的 iPhone > Whisper` 后即可在 App 内浏览和播放。
- 以系统播放器能力为基础，保持轻量、稳定、低维护成本。

## 已实现功能

### 本地资料库

- 直接读取 `我的 iPhone > Whisper` 文件夹。
- App 内文件列表与 Files 文件夹保持同步，外部增删改后会自动刷新。
- 支持文件夹进入、返回、新建、复制、剪切、粘贴、删除和多选管理。
- 搜索页可按音频、视频、字幕、文件夹等类型过滤和查找。

### 播放体验

- 支持本地音频和视频播放。
- 支持播放、暂停、进度拖动、音量调节和播放速度调节。
- 支持快退 / 快进，默认适配听音声常用的小步回退。
- 支持顺序循环、单个循环和随机播放。
- 记忆播放进度，方便继续收听长篇作品。
- 支持睡眠定时。
- 支持后台播放音频会话配置。
- 支持锁屏和耳机远程控制。

### 字幕能力

- 支持外挂字幕加载。
- 支持 SRT、WebVTT、LRC、ASS / SSA 等常见字幕文本格式。
- 支持字幕自动匹配同名或语言提示文件。
- 支持字幕实时显示、开关、时间偏移微调。
- 支持字幕字体、样式、描边、阴影、字号比例和垂直位置设置。
- 支持字幕转语音相关设置入口。

### 界面设计

- 使用 SwiftUI 构建原生 iOS 界面。
- 采用底部迷你播放器和完整播放页。
- 资料库滚动时底部控制区会收缩，减少遮挡。
- 视频播放支持适应、填充和裁剪显示模式。
- 提供浅色、深色和跟随系统主题。

## 运行项目

1. 用 Xcode 打开 `AsmrPlayer.xcodeproj`。
2. 选择 `AsmrPlayer` scheme。
3. 选择 iPhone 模拟器或真机运行。

真机运行时需要在 Xcode 的 Signing & Capabilities 中选择你的开发团队。仓库默认使用占位 bundle id `com.example.Whisper`，请按自己的账号需要修改。

## 使用方式

1. 首次运行 App 后，在 iOS Files 中找到 `我的 iPhone > Whisper`。
2. 将音频、视频和字幕文件放入该目录，也可以自行创建子文件夹整理作品。
3. 回到 App 的资料库页面，点击媒体文件开始播放。
4. 如果字幕文件与媒体文件同名或带有语言标记，Whisper 会尝试自动加载；也可以手动点击字幕文件加载。

## 支持格式

媒体格式主要依赖 iOS 系统播放器能力：

- 音频：`mp3`、`m4a`、`wav`、`aiff`、`flac`、`aac`、`ogg`
- 视频：`mp4`、`m4v`、`mov`
- 字幕：`srt`、`vtt`、`lrc`、`ass`、`ssa`、`txt`

部分封装或编码格式能否播放取决于系统支持情况。
